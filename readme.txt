Для слияния с веткой мастер:
git checkout 201-352_Evseev
git merge -Xignore-space-change master

Для работы с репозиторием:
git clone <ссылка> - клонирует репозиторий
git checkout -b <название новой ветки> - создает новую ветку и переключается на нее
git pull - забирает изменение с git
git add * - сохраняет все изменения
git commit -m "<что-то>" - коммитит изменения
git push - заливает изменения на git
git branch - посомтреть ветки
